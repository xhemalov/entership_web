# Entership - Landing Page

Moderní landing page pro online kurzy umělé inteligence s výrazným fialovým designem a hlavní barvou pozadí #4A044E.

## Funkce

- **Responzivní design** - Optimalizováno pro všechna zařízení
- **Moderní UI** - Využívá Tailwind CSS s custom color scheme
- **Interaktivní komponenty** - FAQ sekce s rozbalovacími položkami
- **Testimonials** - Hodnocení od spokojených studentů
- **CTA sekce** - Jasné výzvy k akci pro konverzi

## Design

- **Hlavní font**: Nasalization (pro nadpisy)
- **Doplňkový font**: Inter (pro text)
- **Barevná paleta**: 
  - Hlavní pozadí: #4A044E (purple-main)
  - Světlejší fialová: #8B5CF6 (purple-light)
  - Tmavší fialová: #3B0764 (purple-dark)
  - Žlutý akcent: #FDE047 (yellow-accent)
- **Styl**: Moderní, technologický vzhled s fialovými odstíny

## Struktura stránky

1. **Navigace** - Logo a hlavní menu s CTA tlačítkem
2. **Hero sekce** - Hlavní nadpis "CHCEŠ SE STÁT MISTREM SVĚTA V AI?" a CTA
3. **Proč sekce** - Důvody proč se učit AI (6 bodů)
4. **Moduly** - Přehled 3 modulů kurzu
5. **Video ukázka** - Placeholder pro video s play tlačítkem
6. **Proces** - 3 kroky k úspěchu (Registrace, Studium, Aplikace)
7. **Obsah kurzu** - Co je součástí (4 hlavní body)
8. **Testimonials** - Hodnocení od 2 spokojených studentů
9. **Finální CTA** - Další video a výzva k akci
10. **FAQ** - 5 často kladených otázek s rozbalovacími odpověďmi
11. **Footer** - Kontaktní informace a odkazy

## Technologie

- Next.js 16
- Tailwind CSS s custom konfigurací
- TypeScript
- React komponenty
- Shadcn/ui komponenty (Card, Accordion, Button)

## Komponenty

- `TestimonialsSection` - Sekce s hodnoceními studentů
- `FAQSection` - Interaktivní FAQ s accordion komponentou
- `FooterSection` - Footer s kontaktními informacemi

## Stav projektu

Kompletní landing page je hotová podle poskytnutého designu. Všechny sekce jsou implementovány s správnou barevností (#4A044E jako hlavní pozadí) a typografií (Nasalization + Inter). Připraveno pro přidání skutečných obrázků, ikon a video obsahu.
